### `suix_getDynamicFieldObject`

Return the dynamic field object information for a specified object

#### Parameters

| Name | Required | Format | Description
|---|---|---|---|
| `parent_object_id` | True | `<ObjectID>` | The ID of the queried parent object |
| `name` | True | `<DynamicFieldName>` | The Name of the dynamic field |

#### Result

##### `<SuiObjectResponse>`

| Name | Required | Format | Description
|---|---|---|---|
| `data` | False | `<[ObjectData]>` | Summary of effects that likely would be generated if the transaction is actually run. Note however, that not all dev-inspect transactions are actually usable as transactions so it might not be possible actually generate these effects from a normal transaction. |
| `error` | False | `<[ObjectResponseError]>` | Execution error from executing the transactions |
