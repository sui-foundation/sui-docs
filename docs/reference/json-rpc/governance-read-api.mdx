### `suix_getCommitteeInfo`

Return the committee information for the asked `epoch`.

#### Parameters

| Name | Required | Format | Description
|---|---|---|---|
| `epoch` | True | `<BigInt_for_uint64>` | The epoch of interest. If None, default to the latest epoch |

#### Result

##### `<DevInspectResults>`

| Name | Required | Format | Description
|---|---|---|---|
| `SuiCommittee` | True | `<CommitteeInfo>` | |

#### Example

Gets committee information for epoch 5000.

```mdx-code-block
<details>
  <summary>
  Request
  </summary>
```

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "suix_getCommitteeInfo",
  "params": [
    "5000"
  ]
}
```
